## Contribution Guidelines

When contributing to this project:

1. **Fork and branch** - Create a feature branch from main
2. **Test thoroughly** - Ensure all changes work as expected
3. **Document changes** - Update README and relevant docs
4. **Submit PR** - Include clear description of changes

## Co-Authored Commits

When pair programming or collaborating, use co-authored commits:

```bash
git commit -m "feat: Description

Co-authored-by: Team Member <email@yoyaku.fr>"
```
